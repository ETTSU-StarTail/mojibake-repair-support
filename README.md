# mojibake-repair-support

文字化け修復支援ツール

## 概要
本プロジェクトは、Python で実装された「文字化け（mojibake）」の検出・修復を支援するツールです。
バイト列の変換や復元処理を通じて、ファイルやテキストデータの文字化け問題を解決します。

## 主な機能
- バイト列から文字列への変換・復元
- 文字化けパターンの検出・解析
- 修復候補の提示
- PySide6 によるGUI拡張も視野

## 使い方
1. 依存パッケージをインストール（uv 管理）
	```
	uv install
	```
2. メインスクリプトの実行
	```
	uv run main.py
	```
3. テストの実行
	```
	uv run test_mojibake.py
	```

## 開発環境
- Python 3.11+
- uv（パッケージ管理）
- PySide6（GUI拡張時）
